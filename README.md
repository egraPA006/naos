# Naos
Not another operating system, simplest version (32-bit x86)

## Description
This a little modified version of the [bare bones tutorial](https://wiki.osdev.org/Bare_Bones).

## Using makefile
Build:
```bash
make build
```
For lsp support run:
```bash
make lsp
```
Run iso with qemu:
```bash
make run
```
